# SootheNotereadme

SootheNote is a calm, private journaling app built to give people a simple, safe place to write and reset.

## Features
- Journaling
- Daily check-ins
- Guided prompts
- Minimal, distraction-free UI

## Run (iOS)
1. Clone this repo
2. Open in Xcode
3. Select a simulator or device
4. Press Run
